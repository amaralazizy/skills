# Skills

Agent skills for daily coding sessions.

## Installation

### npm

```bash
npx skills@latest add amaralazizy/skills/<skill>
```

### pnpm

```bash
pnpm dlx skills@latest add amaralazizy/skills/<skill>
```

### yarn

```bash
yarn dlx skills@latest add amaralazizy/skills/<skill>
```

### bun

```bash
bunx skills@latest add amaralazizy/skills/<skill>
```

## Available Skills

| Skill | Description | Install |
| ----- | ----------- | ------- |
| [nextjs-folder-structure](./nextjs-folder-structure/) | Design and organize Next.js project folder structures following App Router conventions | `pnpm dlx skills@latest add amaralazizy/skills/nextjs-folder-structure` |
| [sprint-prioritization](./sprint-prioritization/) | Use when prioritizing tasks for a sprint, handling stakeholder demands vs team capacity | `pnpm dlx skills@latest add amaralazizy/skills/sprint-prioritization` |
