# Skills

Agent skills for daily coding sessions.

## Installation

### npm

```bash
npx skills@latest add alazizy/skills/<skill>
```

### pnpm

```bash
pnpm dlx skills@latest add alazizy/skills/<skill>
```

### yarn

```bash
yarn dlx skills@latest add mattpocock/skills/<skill>
```

### bun

```bash
bunx skills@latest add mattpocock/skills/<skill>
```
